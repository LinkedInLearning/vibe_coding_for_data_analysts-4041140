## Course Description

Get up to speed with vibe coding for analytics tasks in this course designed for data analysts. Instructor Seth Berry covers the essentials of setting up your environment for effective vibe coding, with the goal of creating an end-to-end application for hosting a machine learning model. From data import to model deployment, Seth highlights which aspects of the process are best left to humans and where LLM assistance can best be leveraged.
